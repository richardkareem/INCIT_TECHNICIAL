## Step 1: Install All Depedency

First you need to install all dependency node modules with 
```bash
# using npm
npm start install

# OR using Yarn
yarn 
```

## Step 2: Start your Application

Running in local emulator or real device
### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
